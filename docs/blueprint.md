# **App Name**: Nigcomsat WiFi Connect - Enugu Airport

## Core Features:

- Landing Page: Landing page with clear branding, service benefits, and a prominent 'Get Free WiFi' button.
- Sign-Up/Sign-In: Sign-up/Sign-in form with email, password, and 'Remember Me' option. Includes 'Forgot Password' functionality.
- Ad Viewing: Simulated ad display section with a 'Proceed to WiFi' button that activates post-viewing.
- Dashboard: WiFi access dashboard with data usage, time remaining, 'Extend Session' (optional) and 'Logout' button.
- Track Data: Tracks users' data consumption.

## Style Guidelines:

- Primary color: #1976D2 (Nigcomsat blue) to convey trust and reliability.
- Background color: #E3F2FD (Light blue from Nigcomsat's palette) as a softer, desaturated tone.
- Accent color: #2962FF (A brighter blue from Nigcomsat's palette) to ensure it stands out.
- Font pairing: 'Poppins' (sans-serif) for headlines, and 'Roboto' (sans-serif) for body text, prioritizing readability and modern aesthetics.
- A clean and modern design ensuring a user-friendly experience. Mobile responsiveness for access on any device.
- Simple and clear icons for navigation and functions within the site, using a consistent style that aligns with the Nigcomsat brand.
- Subtle transitions and animations for a smoother user experience (e.g., button hover effects, ad completion feedback), enhancing engagement without being distracting.
